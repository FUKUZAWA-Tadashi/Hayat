Bool

論理値クラス。 true と false の2種類のインスタンスしか無い。
newによるインスタンス生成はできない。

親クラス

Object

演算子

! (単項演算子) 論理否定
== 比較：等しい時に true
&& 論理積
通常 && 演算子は、コンパイラ内部で処理され、この 演算子実装を呼び出さない。敢えてメソッドとして呼び出したい場合 には、 x.getMethod(:"&&").call(y) という様に して使う。
|| 論理和
通常 || 演算子は、コンパイラ内部で処理され、この 演算子実装を呼び出さない。敢えてメソッドとして呼び出したい場合 には、 x.getMethod(:"||").call(y) という様にして使う。

備考

|| , && 演算は 演算子ショートカットのため、通常はコンパイラ内部で処理される。
例えば、xがtrueの時、 x || f() という演算では、 結果が true になる筈なので、 f() を計算しない。 敢えて f() を計算した上で論理和の計算を行ないたい場合には、 y = f(); x || y としたり、 x.getMethod(:"||").call(f()) とする。
