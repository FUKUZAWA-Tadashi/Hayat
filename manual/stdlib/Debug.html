標準ライブラリ stdlib

Debug

Debug用クラスメソッド群。 デバッグ表示は、デバッグビルドしたエンジンでしか表示できないので注意。
newによるインスタンス生成はできない。

クラスメソッド

Debug.nop()
何もしない。 C++のデバッガで、 HSca_Debug.cpp のこのメソッドの定義部分に ブレークポイントを仕掛けるといった用途に使える。
Debug.debug?()
デバッグバージョンとしてビルドされたエンジンかどうか
Debug.p(...)
引数の値と型をデバッグ出力する。
デバッグバージョンでなければ何もしない。
Debug.assert(x)
引数がfalseだとエラー停止する。
デバッグバージョンでなければ何もしない。
Debug.assertMsg(...)
第一引数がfalseだと、残り引数をデバッグ出力してエラー停止する。
デバッグバージョンでなければ何もしない。
Debug.print(...)
引数をデバッグ出力する。
デバッグバージョンでなければ何もしない。
sayCommandStart(x,n), sayCommand(x), sayCommandEnd()
デバッグ出力する。 
// 以下のように使う。
Debug「x = {x}, y = {y}」
デバッグバージョンでなければ何もしない。
Debug.setPrintOpcode(b)
VM命令1ステップ毎にバイトコードをデバッグ表示するかどうかを 設定する。 true で表示設定、false で非表示設定。
デバッグバージョンでなければ設定しても何も表示しない。
Debug.setPrintStack(b)
VM命令1ステップ毎にVMのスタック状態をデバッグ表示するか どうかを設定する。 true で表示設定、false で非表示設定。
デバッグバージョンでなければ設定しても何も表示しない。
Debug.printAllStackTrace()
全スレッドのスタックトレースを表示する。
デバッグバージョンでなければ何もしない。
Debug.printGCPhase()
ガーベジコレクションの進行状況をデバッグ表示する。
デバッグバージョンでなければ何もしない。
Debug.printGCMark(val)
オブジェクトvalのガーベジコレクション用markフラグの状況をデバッグ表示する。
デバッグバージョンでなければ何もしない。
