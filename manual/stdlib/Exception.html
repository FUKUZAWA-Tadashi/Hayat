標準ライブラリ stdlib

Exception

例外クラス。
例外オブジェクトは、1つの値と例外タイプシンボルを持つ。 例外の種類は、例外タイプによって区別する。他の言語と異なり、 継承を使用していない。

親クラス

Object

コンストラクタ

new Exception(val = nil, type = :exception)
新しい例外オブジェクトを生成する。
保持するオブジェクトと、例外タイプシンボルを指定できる。 例外タイプを省略すると、 :exception となる。 保持するオブジェクトを省略すると、 nil となる。

インスタンスメソッド

type()
例外タイプシンボルを取得
setVal(v)
保持するオブジェクトを設定する。
val()
保持するオブジェクトを取得。
getLabel()
typeが :goto の時のみ、飛び先ラベルを取得。そうでなければ nil。
goto 命令は、catch によって捕捉される。その時の例外タイプは :goto となっている。 保持するオブジェクトは、goto直前に実行していた文の値である。
backTraceSize()
backtraceの個数を取得。
backTraceStr(i)
i番目のbacktrace情報をStringBufferで取得する。 (0 <= i < backTraceSize())
i = 0 が一番深いレベルでExceptionが発生した場所。
i が範囲を越えていたら nil が返る。
