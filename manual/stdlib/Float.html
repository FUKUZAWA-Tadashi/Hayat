- (単項演算子) 単項マイナス

* 乗算
/ 除算
** べき乗
== 比較：等しい時に true
<> 比較：等しくない時に true
!= 比較：等しくない時に true
<=> 比較：左辺が小さい時に -1, 大きい時に 1, 同じ場合は 0
< 比較：左辺が小さい時に true
<= 比較：左辺が小さいか等しい時に true
> 比較：左辺が大きい時に true
>= 比較：左辺が大きいか等しい時に true
