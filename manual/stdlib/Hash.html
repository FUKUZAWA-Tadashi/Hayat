Hash

連想配列。

親クラス

Object

コンストラクタ

new Hash(bucketSize, loadFactor)
新しいハッシュを作成する。 bucketSizeは初期容量で、デフォルト値は 31 。 loadFactorは、内容物の個数がbucketSizeに対してこの比率を越えた 時、bucketSizeを増やしれrehashする数値。デフォルト値は 0.75 。

演算子

== 比較：全内容が等しい時に true
[] キーに関連づけられた値を取り出す。
[]= キーに対して値を関連づける。

インスタンスメソッド

size()
入っているデータの個数
capacity()
バケットの容量
keys()
全keyを配列に入れる
values()
関連付けられた全要素を配列に入れる
rehash()
ハッシュ再計算再構築
ハッシュのキーにしたオブジェクトが、外部で変化させられて hashCode値が変わってしまう場合がある。そうすると、 [] 演算子によって検索できなくなってしまう。 rehash()を実行すると、全てのキーオブジェクトのhashCodeを再計算 して格納し直すので、再び検索できるようになる。
