abs()

times()
数値の回数だけ繰り返すFiberを作る。 yieldする値は、0 から self-1 まで。
times(closure)
数値の回数だけ繰り返して closure を呼ぶ。 closureの引数として、 0 から self-1 までの値を渡す。 
5.times{|i| a[i] = i * 10}
times().iterate(closure)より高速。
upto(max)
selfからmaxまでの値をyieldするFiberを作る。
upto(max, closure)
selfからmaxまでの値をclosureの引数として渡しながら繰り返す。 
5.upto(7){|i| x += a[i]}  // x に a[5],a[6],a[7] の値を加算する
upto(max).iterate(closure)より高速。
