標準ライブラリ stdlib

Math

数値演算関数群。
newによるインスタンス生成はできない。

定数

E 自然対数の底。 2.7182818284590452354
PI 円周率。 3.14159265358979323846

クラスメソッド

Math.sin(x)
正弦関数。 x は Float で、戻り値も Float である。
Math.cos(x)
余弦関数。 x は Float で、戻り値も Float である。
Math.sqrt(x)
平方根。 x は Float で、戻り値も Float である。
Math.log(x)
自然対数。 x は Float で、戻り値も Float である。
