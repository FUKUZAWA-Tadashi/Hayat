標準ライブラリ stdlib

Method

レシーバとメソッドをペアにして保持するオブジェクト。

親クラス

Object

コンストラクタ

new Method(obj, methodSymbol)
新しいMethodオブジェクトを作成する。
objはメソッドを実行するインスタンス。 methodSymbol は、 メソッド名を表わすシンボル。

演算子

() メソッド呼び出し。 call() と同じ。

インスタンスメソッド

call(...)
メソッド呼び出し。
x = new Method(a, :foo)
x.call(1,2)
// これは、 a.foo(1,2) を実行する。
