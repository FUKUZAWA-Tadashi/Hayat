標準ライブラリ stdlib

Object

Objectは、全てのクラスの親クラスとなる。 なので、どのオブジェクトもこのクラスのメソッドを呼ぶ事ができる。

インスタンスメソッド

getClass()
オブジェクトのクラスを返す
classSymbol()
オブジェクトのクラスのシンボルを返す
concatToStringBuffer(sb,mode)
StringBufferオブジェクトsbの末尾に、文字列表現を追加する
hashCode()
Hashのキーとなるのに必要なハッシュ値を計算
sameObject?(x)
参照レベルで self と x が同じオブジェクトかどうか調べる 
x = [1,2,3]
y = x
z = [1,2,3]
である時、
x.sameObject?(y) は true
x.sameObject?(z) は false
x == y は true
x == z は true
となる。
getMethod(methodSymbol)
メソッドシンボルから、Methodオブジェクトを作成して返す 
a = []
f = a.getMethod(:push)
f(100)    // a.push(100)  =>  a == [100]
f(99)     // a.push(99)   =>  a == [100,99]
haveMethod?(methodSymbol)
self がメソッド methodSymbol を持っているかを調べる 
a = [1]
a.haveMethod?(:foobar)  // => false  Arrayクラスはfoobarメソッドを持っていない
callIfMethodExist(methodSymbol, *args)
もしselfがメソッドmethodSymbolを持っているならば、 引数argsを渡して呼び出す。 メソッドを持っていなければ、 nil を返す。

演算子

== 内容が一致するか比較する。一致していれば true
<> 内容が一致しなければ true。 ==の否定。 子クラスは、 == を実装すれば <> を使える。
!= 内容が一致しなければ true。 <>と同じ。 子クラスは、 == を実装すれば != を使える。
