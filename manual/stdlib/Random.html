標準ライブラリ stdlib

Random

乱数生成クラス。 32bitのM系列乱数(ガロアLFSR)である。
newによるインスタンス生成はできない。

クラスメソッド

Random.srand(x)
乱数シードを整数xで初期化。戻り値はx。
Random.seed()
現在の乱数シードを返す。
Random.rand()
乱数を生成し、整数を返す。値の範囲はIntの全範囲。
Random.rand(x)
0 <= rand(x) < x の範囲の整数乱数を生成する。
Random.rand(x,y)
x <= rand(x,y) <= y の範囲の整数乱数を生成する。
